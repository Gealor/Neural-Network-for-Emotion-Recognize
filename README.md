# Цель проекта
Написать собственную нейронную сеть для задач распознавания звуков.
В данной ветке реализована сверточная нейронная сеть(CNN) с 3 скрытыми слоями.
В репозитории представлено две ветки, первая cnn просто сверточная нейронная сеть без аугментирования данных и вторая ветка cnn_with_augmented_data с аугментированием данных

# Описание
В проекте представлено три python скрипта:
- download_dataset: для скачивания датасета RAVDRESS(https://www.kaggle.com/datasets/uwrfkaggler/ravdess-emotional-speech-audio?resource=download)
- prepare_data: для подготовки данных для обучения модели(с аугментацией)
- model_training: хранится непосредственная архитектура модели, а так же визуализация результатов обучения

# Результаты
Модель прогонялась при следующих параметрах:
1. Количество эпох=200

По результатам обучения можно сделать вывод, что сверточная нейронная сеть, которая принимает на вход двумерную mfcc спектрограмму справляется лучше с классификацией звука, по сравнению с простой mlp, но прослеживаются признаки переобучения модели(это хорошо видно по графикам точности и потерь) 
С графиками можно ознакомиться в папке 200_epochs_with_augmented_data

# Предполагаемые причины: 
1. Отсутствие нормализации/скалирования данных перед тем как подавать их модель
2. Попробовать регуляризацию скорости обучения по мере приближения к концу

# Предложения
1. Использовать StandardScaler из sklearn, для нормализации данных
2. Использовать ReduceLROnPlateau (снижение LR, если валидационная потеря не уменьшается)
3. Использовать LSTM/GRU для учета временной последовательности
