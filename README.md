# Цель проекта
Написать собственную нейронную сеть для задач распознавания звуков.
В данной ветке я проверяю новую архитектуре CRNN.

# Описание
В проекте представлено три python скрипта:
- download_dataset: для скачивания датасета RAVDRESS(https://www.kaggle.com/datasets/uwrfkaggler/ravdess-emotional-speech-audio?resource=download)
- prepare_data: для подготовки данных для обучения модели(с аугментацией)
- model_training: хранится непосредственная архитектура модели, а так же визуализация результатов обучения

# ЗАМЕЧАНИЯ
В нативной Windows системе отсутствует поддержка обучения на CUDA ядрах, для обучения на видеокарте я использую WSL2. Гайд о том как настроить WSL2 для машинного обучения на видеокарте: https://www.youtube.com/watch?v=402DciWGvt8

# Результаты
Модель прогонялась при следующих параметрах:

С результатами обучения со всеми описанными изменениями можно ознакомиться в папке results_crnn.

По результатам обучения можно сказать, что проблема переобучения вернулась и валидационные потери не опускаются ниже 1.6, а валидационная точность не повышалась выше 0.550

# Предложения для исправления
1. Добавить dropout'ы(dropout и recurrent_dropout) в GRU слой
2. Понизить количество units в GRU слое(например со 128 до 64)
3. Понизить скорость обучения(например до 0.0005)
4. Убрать один сверточный слой(4-ый)
5. Добавить Dropout'ы в конце каждого сверточного блока
6. Сделать двунаправленный RNN слой(использовать layers.Bidirectional)

# Промежуточные результаты 1
