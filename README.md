# Цель проекта
Написать собственную нейронную сеть для задач распознавания звуков.
В данной ветке реализована простейшая полносвязная модель(MLP) с 1 скрытым слоем.

# Описание
В проекте представлено три python скрипта:
- download_dataset: для скачивания датасета RAVDRESS(https://www.kaggle.com/datasets/uwrfkaggler/ravdess-emotional-speech-audio?resource=download)
- prepare_data: для подготовки данных для обучения модели
- model_training: хранится непосредственная архитектура модели, а так же визуализация результатов обучения

# Результаты
Модель прогонялась два раза при следующих параметрах:
1. Количество эпох=10
2. Количество эпох=200

По результатам обучения можно сделать вывод, что простейшая полносвязная нейронная сеть, которая принимает на вход сплющенную mfcc спектрограмму плохо справляется с классификацией звука. 

# Предполагаемые причины: 
1. Отсутствие нормализации/скалирования данных перед тем как подавать их модель
2. Архитектура модели слишком простая и одного скрытого слоя банально недостаточно, чтобы сделать корректировку весов.

# Предложения
1. Перейти от MLP к CNN архитектуре, т.к. в CNN архитектура лучше работает с изображениями и матрицами MFCC
2. Отфильтровать датасет, т.е. убрать музыкальные данные, т.к. в музыке понятие эмоций очень субъективно
3. Использовать LSTM/GRU для учета временной последовательности
